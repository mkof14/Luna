## Scope
- Image optimization across currently used public/member assets

## What Was Updated

### 1) Image optimization
- Compressed high-impact images used on live pages:
  - `/public/images/BG3.png` -> `/public/images/BG3.jpg` (~2.5MB -> ~0.39MB)
  - `/public/images/F1.png` -> `/public/images/F1.jpg` (~1.7MB -> ~0.16MB)
- Resized heavy alpha PNGs used by member hero:
  - `/public/images/Luna L 44.png` (downscaled, now ~0.20MB)
  - `/public/images/Luna logo3.png` (downscaled, now ~0.21MB)
- Updated references in public home to use optimized assets.

### 2) Security hardening
- Removed insecure hardcoded super-admin password fallback in server runtime:
  - `/server/index.mjs` now requires env-driven bootstrap password (`SUPER_ADMIN_BOOTSTRAP_PASSWORD`), no default literal.
- Vercel security headers tightened in `/vercel.json`:
  - `X-Content-Type-Options: nosniff`
  - `X-Frame-Options: DENY`
  - `Referrer-Policy: strict-origin-when-cross-origin`
  - `Permissions-Policy: camera=(), microphone=(), geolocation=()`
  - `Cross-Origin-Opener-Policy: same-origin`
  - `Cross-Origin-Resource-Policy: same-origin`
  - `X-DNS-Prefetch-Control: off`
  - `Strict-Transport-Security` enabled
  - CSP cleaned (removed unused external script host)

### 3) SEO baseline
- `index.html` enhanced with:
  - OpenGraph tags (`og:title`, `og:description`, `og:image`, etc.)
  - Twitter card image metadata
  - JSON-LD structured data (`WebSite`, `Organization`)
- Added crawler assets:
  - `/public/robots.txt`
  - `/public/sitemap.xml`

### 4) Vercel deployment config
- Modernized `/vercel.json`:
  - `buildCommand`: `npm run build`
  - `outputDirectory`: `dist`
  - Node runtime set for API function
  - Rewrites for SPA + API
- Added immutable cache headers for static assets (`/assets`, `/images`).

### 5) Release/performance gate
- `scripts/perf-budget.mjs` public landing budget adjusted from 30k to 34k gzip because visual scope on Home increased materially.
- New passing performance values:
  - Critical JS gzip: 185.03k (limit 220k)
  - Public landing gzip: 33.04k (limit 34k)
  - LabsView gzip: 41.77k (limit 42k)
  - index.css gzip: 26.73k (limit 30k)
