# Luna Store Assets Checklist

## App identity
- [ ] Final app icon (1024x1024 source)
- [ ] Splash image aligned with Luna brand
- [ ] Adaptive icon layers for Android verified

## App Store screenshots (iOS)
- [ ] 6.7" iPhone screenshots (required set)
- [ ] 6.5"/6.1" fallback if needed
- [ ] Screens include Today, Voice Reflection, Reflection Result, Rhythm
- [ ] No placeholder text or debug visuals

## Google Play screenshots
- [ ] Phone screenshots (minimum required count)
- [ ] Optional tablet screenshots (if supporting tablet-focused listing)
- [ ] Feature graphic (1024x500)

## Content quality
- [ ] All visible copy localized and consistent with app language
- [ ] No lorem ipsum
- [ ] No references to unfinished features

## Privacy / policy
- [ ] Privacy policy page is public and accessible
- [ ] Contact/support page is public and accessible
- [ ] Data safety answers match implemented behavior
